Help

How to Use & Data Schemas

Data never leaves your browser. Import with CSV or JSON bundle.

  • Branding: Upload logo (or URL) → Save Branding. “Clear Logo” resets to default; “Reset Brand” clears both.
  • Slicers: Add custom values, Select All, Delete; “Reset Options From Data” rebuilds the lists.
  • OBE: Provide real OBE mapping % by dept/program/year in its dataset (no more proxy needed).
CSV columns 
placements: program, batch_year, placed_count, total_count, median_ctc_inr, internship_pct, dept
research: title, type, citations, year, dept, program, link
budget: item, type (Capex/Opex), amount_inr, year, owner, project, status (planned/approved/spent), dept
consultancy: client, value_inr, start_date (YYYY-MM-DD), end_date, owner_dept, status (pipeline/active/closed)
obe: dept, program, year, obe_mapped_pct
Privacy: Everything persists in localStorage. Use Import/Export to move devices.
